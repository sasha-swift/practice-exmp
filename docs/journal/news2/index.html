Промежуточные результаты проекта

18.10.2024

Разработка диаграмм взаимодействия процессов

В проекта работы были разработаны диаграммы, отражающие последовательность и взаимосвязь ключевых процессов, задействованных в проверке остаточных знаний студентов. Диаграммы визуализируют:

  • Инициацию проверки — запуск процесса со стороны администратора или ЛПР (лица, принимающего решение)

  • Формирование выборки дисциплин и преподавателей — логика выбора на основе заданных критериев

  • Подготовку и проведение тестирования — автоматизация этапов уведомления, доступа к тестам и сбора результатов

  • Обработку и анализ результатов — последовательность действий по оценке, формированию отчётов и их передаче заинтересованным сторонам

Данные диаграммы обеспечивают структурированное понимание логики, способствуют формализации требований к системе и служат основой для проектирования backend- и frontend-частей. Их наличие упрощает коммуникацию и позволяет заранее выявить потенциальные узкие места и зависимости между этапами.

Разработка ER-модели базы данных

Создана ER-модель, отражающая структуру данных системы: определены основные сущности (пользователи, дисциплины, студенты, тесты, отчёты) и связи между ними. Модель послужила основой для построения базы данных, обеспечив целостность, масштабируемость и поддержку бизнес-логики проекта.

Составление диаграммы формирования списка дисциплин для тестирования

Составлена диаграмма, отражающая логику отбора дисциплин для последующего тестирования. Визуализированы ключевые этапы процесса, задействованные роли и предполагаемые взаимодействия с системой. Диаграмма послужила основой для будущей реализации функционала и уточнения требований к разработке.

Разработка фреймов интерфейса для ролей ЛПР и Преподаватель

Разработаны фреймы пользовательского интерфейса для ролей ЛПР и преподавателя, отражающие структуру будущих экранов и их ключевые элементы. Фреймы учитывают различия в функциональности и доступе, помогая определить логику взаимодействия с системой. Они стали основой для последующего дизайна и реализации интерфейсов.
