Итоги реализации проекта по состоянию на вторую аттестационную проверку

05.12.2024

Используемые технологии

Для разработки проекта выбраны следующие технологии: C# — для создания backend-части и логики API, React — для построения интерфейсов пользователя, MySQL — в качестве системы управления базами данных, и Git — для контроля версий и совместной работы над кодом. Эти инструменты используются на текущем этапе разработки и обеспечивают основу для реализации всех компонентов системы.

Создание списка API-методов для роли ЛПР

Сформирован предварительный список API-методов, необходимых для реализации функционала роли ЛПР. Перечень охватывает основные операции взаимодействия с данными, включая получение отчётов, управление дисциплинами и доступом к результатам тестирования. Список послужит основой для последующей реализации и уточнения требований.

Написание SQL-запросов

Подготовлены SQL-запросы для обработки ключевых показателей: подсчёт студентов в группе, определение прошедших и не прошедших тестирование, выявление недопущенных и неявившихся, а также получение результатов промежуточной аттестации и итогового тестирования. Запросы лягут в основу аналитического блока системы.

Разработка дизайна интерфейса системы

Создан прототип пользовательского интерфейса для роли заведующего кафедрой и доработан прототип для роли ЛПР с учётом новой логики и визуальных требований. Эти решения формируют основу для последующего этапа реализации интерфейса системы.

Разработка и размещение сайта

Создан сайт на зарегистрированном домене с подключением к хостингу. Выполнено подключение базы данных к backend-приложению и реализованы API-методы для отображения данных на главной странице.
